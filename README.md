# Sales Data Analysis Project

## Overview
This project analyzes sales data using Python and Power BI.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Power BI

## Features
- Data Cleaning
- Revenue Analysis
- Monthly & Yearly Trends
- Dashboard Visualization

## Insights
- Identified highest revenue year
- Analyzed monthly trends
- Found correlation between orders and revenue.
